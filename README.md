# BoardGame

Premium 3D Board Game Collection built with Three.js and Vite.

## Games

- **Tic Tac Toe** — Connect 3 in a row to win
- **Checkers** — Jump to capture, reach the end to become King
- **Chess** — Full legal moves, capture, pawn promotion
- **Ludo** — Roll dice to race tokens home
- **Monopoly** — Roll and travel around the board
- **Backgammon** — Bear off all checkers to win
- **Mill (Nine Men's Morris)** — Form 3 in a row to remove opponent pieces
- **Cards** — Hearts card display
- **Dominoes** — Tile showcase

## Tech Stack

- [Three.js](https://threejs.org/) — 3D rendering
- [Vite](https://vitejs.dev/) — Build tool
- KayKit Board Game Assets (GLTF)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
